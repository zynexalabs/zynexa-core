# Changelog

All notable changes to Zynexa are documented in this file.

## [0.1.0] - 2025-11-24

### Added
- Zero-knowledge identity system with Ed25519 keypairs
- BIP-39 mnemonic seed phrase generation
- End-to-end encrypted messaging framework
- Private encrypted file storage UI
- DEX swap interface with stealth transaction support
- Browser-based Web3 module integration
- GitHub Actions CI/CD workflow
- Security policy and vulnerability reporting guidelines
- Environment configuration template
- Smoke test validation script

### Infrastructure
- Monorepo structure (client/server/shared)
- TypeScript configuration for full-stack development
- Vite dev server with hot module replacement
- Tailwind CSS with Radix UI components
- Drizzle ORM for database management
- Express.js backend framework

### Documentation
- Comprehensive README with feature roadmap
- SECURITY.md with vulnerability disclosure process
- CONTRIBUTING.md for community guidelines
- Code of Conduct
- This CHANGELOG
